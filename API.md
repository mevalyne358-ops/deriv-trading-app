# API Documentation\n\n## Deriv Trading Platform API\n\n### Base URLs\n- **Demo**: `wss://ws.derivws.com/websockets/v3?app_id=YOUR_APP_ID`\n- **Live**: `wss://ws.deriv.com/websockets/v3?app_id=YOUR_APP_ID`\n\n---\n\n## Authentication\n\n### Authorize\nAuthorize with API token to access trading features.\n\n```json\n{\n  \"authorize\": \"YOUR_API_TOKEN\"\n}\n```\n\n**Response**:\n```json\n{\n  \"authorize\": 1,\n  \"account_list\": [],\n  \"loginid\": \"CR123456\",\n  \"is_virtual\": true\n}\n```\n\n---\n\n## Account Operations\n\n### Get Balance\n```json\n{\n  \"balance\": 1,\n  \"subscribe\": 1\n}\n```\n\n**Response**:\n```json\n{\n  \"balance\": 10000.00,\n  \"currency\": \"USD\",\n  \"loginid\": \"CR123456\"\n}\n```\n\n### Get Account Status\n```json\n{\n  \"get_account_status\": 1\n}\n```\n\n---\n\n## Market Data\n\n### Subscribe to Ticks (Real-time Prices)\n```json\n{\n  \"ticks\": \"EURUSD\",\n  \"subscribe\": 1\n}\n```\n\n**Response**:\n```json\n{\n  \"tick\": {\n    \"bid\": 1.08500,\n    \"ask\": 1.08505,\n    \"epoch\": 1634567890,\n    \"quote\": 1.08502,\n    \"symbol\": \"EURUSD\"\n  }\n}\n```\n\n### Subscribe to Candles\n```json\n{\n  \"ticks_history\": \"EURUSD\",\n  \"style\": \"candles\",\n  \"granularity\": 60,\n  \"count\": 100,\n  \"subscribe\": 1\n}\n```\n\n**Parameters**:\n- `granularity`: 60 (1 min), 300 (5 min), 900 (15 min), 3600 (1 hour), etc.\n\n---\n\n## Trading\n\n### Get Contract Proposal\n```json\n{\n  \"proposal\": 1,\n  \"subscribe\": 1,\n  \"symbol\": \"EURUSD\",\n  \"contract_type\": \"CALL\",\n  \"duration\": 5,\n  \"duration_unit\": \"m\"\n}\n```\n\n**Response**:\n```json\n{\n  \"proposal\": {\n    \"ask_price\": 25.50,\n    \"bid_price\": 24.50,\n    \"display_value\": \"$25.50\",\n    \"id\": \"1234567890\",\n    \"longcode\": \"Win payout if EUR/USD is strictly higher than 1.08550 at 15:30:00 GMT.\",\n    \"payout\": 50,\n    \"spot\": 1.08500,\n    \"spot_time\": 1634567890,\n    \"symbol\": \"EURUSD\",\n    \"type\": \"CALL\"\n  }\n}\n```\n\n### Buy Contract\n```json\n{\n  \"buy\": \"1234567890\",\n  \"price\": 1\n}\n```\n\n**Response**:\n```json\n{\n  \"buy\": {\n    \"balance_after\": 9974.50,\n    \"contract_id\": 98765432,\n    \"longcode\": \"Win payout if EUR/USD is strictly higher than 1.08550 at 15:30:00 GMT.\",\n    \"payout\": 50,\n    \"purchase_time\": 1634567890,\n    \"shortcode\": \"CALL_EURUSD_5_1.08550_1634567890S0_1634567895\",\n    \"transaction_id\": 12345\n  }\n}\n```\n\n### Close Contract\n```json\n{\n  \"sell\": \"98765432\",\n  \"price\": 1\n}\n```\n\n---\n\n## Portfolio\n\n### Get Open Positions\n```json\n{\n  \"portfolio\": 1\n}\n```\n\n**Response**:\n```json\n{\n  \"portfolio\": {\n    \"contracts\": [\n      {\n        \"contract_id\": 98765432,\n        \"symbol\": \"EURUSD\",\n        \"longcode\": \"Win payout if EUR/USD is strictly higher than 1.08550 at 15:30:00 GMT.\",\n        \"payout\": 50,\n        \"purchase_time\": 1634567890\n      }\n    ]\n  }\n}\n```\n\n### Get Transaction History\n```json\n{\n  \"statement\": 1,\n  \"limit\": 50\n}\n```\n\n---\n\n## Contract Types\n\n- `CALL`: Price will be higher at expiry\n- `PUT`: Price will be lower at expiry\n- `CALLE`: Call with barrier above spot\n- `PUTE`: Put with barrier below spot\n\n## Duration Units\n\n- `m`: Minutes\n- `h`: Hours\n- `d`: Days\n- `t`: Ticks\n\n## Symbols\n\n### Forex\n- EURUSD, GBPUSD, AUDUSD, NZDUSD, USDJPY, USDCAD, USDCHF\n\n### Synthetics\n- R_100, R_75, R_50, R_25, RDBULL, RDBEAR, CRASH_500, CRASH_1000\n\n### Commodities\n- XAUUSD (Gold), XAGUSD (Silver), WTIUSD (Crude Oil), BRNUSD (Brent Oil)\n\n### Indices\n- VIX (Volatility Index)\n\n---\n\n## Error Handling\n\nAll error responses include an `error` field:\n\n```json\n{\n  \"error\": {\n    \"code\": \"AuthorizeError\",\n    \"message\": \"Authorization token is invalid.\"\n  }\n}\n```\n\n### Common Errors\n- `InvalidToken`: API token is invalid or expired\n- `InsufficientBalance`: Not enough balance to place trade\n- `InvalidSymbol`: Symbol doesn't exist\n- `InvalidDuration`: Duration is not allowed\n\n---\n\n## Rate Limits\n\n- No official rate limits for demo\n- Live trading has request throttling\n- Keep-alive ping every 30 seconds\n\n---\n\n## Examples\n\n### Complete Trading Flow\n\n```javascript\n// 1. Connect and authorize\nawait api.connect();\nawait api.authorize(token);\n\n// 2. Get market data\napi.subscribeTicks('EURUSD', (tick) => {\n  console.log(`Current price: ${tick.quote}`);\n});\n\n// 3. Get contract proposal\nconst proposal = await api.getContractProposal({\n  symbol: 'EURUSD',\n  contract_type: 'CALL',\n  duration: 5,\n  duration_unit: 'm'\n});\n\n// 4. Buy contract\nconst buyResponse = await api.buyContract(proposal.proposal.id);\nconsole.log(`Trade ID: ${buyResponse.buy.contract_id}`);\n\n// 5. Monitor position\nawait new Promise(resolve => setTimeout(resolve, 300000)); // Wait 5 minutes\n\n// 6. Close trade\nconst closeResponse = await api.closePosition(buyResponse.buy.contract_id);\nconsole.log(`Payout: ${closeResponse.sell.payout}`);\n```\n\n---\n\nFor more details, visit: https://developers.deriv.com/\n
